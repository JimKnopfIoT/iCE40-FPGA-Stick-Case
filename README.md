# iCE40-FPGA Stick enclosure
3D printable enclosure for iCE40-FPGA Stick, created with Rhino 6.

#### 3D Case
The enclosure consists of 2 parts which are clamped together (no screws). 

Requirements:
* 3D printer 
* Filament of your choice
